# Experiment-1-OOPS

## Day Of Week
This program prompts the user to enter a number between 1 and 7 and returns the corresponding day of the week.

### Usage
1. Compile the Java file:
   ```
   javac DayOfWeek.java
   ```
2. Run the application:
   ```
   java DayOfWeek
   ```
